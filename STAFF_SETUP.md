# 👤 คู่มือพนักงาน — เชื่อมต่ออีเมล ASEFA Mail (ครั้งแรกครั้งเดียว)

ทำตามนี้ครั้งเดียวตอนใช้งานครั้งแรก ใช้เวลา ~5 นาที

> เปิดโปรแกรม **ASEFA Mail** → หน้าต่าง "เชื่อมต่อ Zoho ครั้งแรก" จะเด้งขึ้นมาเอง
> ทำตามขั้นตอนในหน้าต่างนั้นคู่กับคู่มือนี้

---

## ① เปิดหน้าตั้งค่า Zoho
ในหน้าต่างของโปรแกรม กดปุ่ม **"① เปิดหน้า Zoho API Console"**
→ เบราว์เซอร์จะเปิดเว็บ → **เข้าสู่ระบบด้วยอีเมล @asefa.co.th ของคุณ**

---

## ② สร้าง "Self Client"
ในเว็บที่เปิดขึ้น:
1. กด **ADD CLIENT** (มุมขวาบน)
2. เลือก **Self Client** → กด **CREATE** → **OK**
3. จะเห็น **Client ID** และ **Client Secret**
4. **คัดลอกทั้งสองค่า** มาวางในโปรแกรม (ช่อง ② และ ③)

---

## ③ ขอรหัส (Authorization Code)
ยังอยู่ในเว็บเดิม:
1. กดแท็บ **Generate Code**
2. ช่อง **Scope** → วางข้อความนี้ (คัดลอกได้จากในโปรแกรม):
   ```
   ZohoMail.messages.ALL,ZohoMail.accounts.READ,ZohoMail.folders.READ
   ```
3. **Time Duration** เลือก **10 minutes**
4. **Scope Description** พิมพ์อะไรก็ได้ เช่น `ASEFA Mail`
5. กด **CREATE** → จะได้ **รหัสยาวๆ** → **คัดลอก**

---

## ④ วางรหัสในโปรแกรม แล้วเชื่อมต่อ
กลับมาที่โปรแกรม ASEFA Mail:
1. วางรหัสในช่อง **④ Authorization Code**
2. เลือกภูมิภาคให้ถูก (ปกติคือ **com**)
3. กดปุ่ม **⑤ เชื่อมต่อ**

✅ ถ้าขึ้น **"🎉 สำเร็จ!"** พร้อมอีเมลของคุณ = เสร็จเรียบร้อย ใช้งานได้เลย!

---

## ❓ ถ้าติดปัญหา

| อาการ | แก้ |
|---|---|
| "แลก code ไม่สำเร็จ / code หมดอายุ" | รหัสใช้ได้แค่ 10 นาที — กลับไปกด CREATE ขอรหัสใหม่ |
| "Account not found" | เลือกภูมิภาคผิด ลองเปลี่ยนเป็น com |
| ไม่เห็นปุ่ม ADD CLIENT | ยังไม่ได้ล็อกอิน หรือล็อกอินผิดบัญชี |

ติดปัญหาอื่นๆ ติดต่อ IT ได้เลยครับ

---

## หลังเชื่อมต่อแล้วทำอะไรได้บ้าง

- 🏠 **หน้าหลัก** — ดูพื้นที่อีเมลว่าใกล้เต็มไหม
- 🔔 **เฝ้าดูเมล** — เด้งเตือนเมื่อมีเมลเข้า-ออก
- 🗄️ **สำรองข้อมูล** — โหลดเมล + ไฟล์แนบเก็บไว้ (HTML หรือ .eml)
- ⚙️ **ตั้งค่า** — แก้ไขค่าต่างๆ ภายหลัง
